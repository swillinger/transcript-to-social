# Platform Specifications

## Twitter / X
- **Max chars per tweet**: 280
- **Thread format**: Tweet 1 announces topic + tags + link, tweets 2-N contain insights
- **End tweet 1 with**: 👇 (thread indicator)
- **Handles**: @username (verify before using — never fabricate)
- **URLs**: count as 23 chars regardless of actual length
- **Hashtags**: 2-3 max, placed at end of tweet or inline naturally
- **Numbering**: optional "1/" style, but keep total tweet count visible

## LinkedIn
- **Max chars**: 3000
- **Format**: Hook line → blank line → body paragraphs → blank line → hashtags
- **Hook**: strong first line (shown before "...see more" fold), no more than 150 chars
- **Hashtags**: 3-5 at the bottom, relevant to industry/topic
- **Mentions**: @FirstName LastName (LinkedIn format, not Twitter @handle)
- **Tone**: professional but conversational; first-person is fine
- **Line breaks**: use blank lines between paragraphs for readability

## Bluesky
- **Max chars per post**: 300
- **Thread format**: same as Twitter — announcement post + insight posts
- **Handles**: @handle.bsky.social or @handle.domain format
- **Hashtags**: supported, 2-3 max
- **URLs**: use full URL (no link shortening needed)
